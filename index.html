    How to use ?

  • Open client 1
  • Click create offer
  • A SDP will be generated in the second text box copy it
  • Open client 2
  • Now paste the copied SDP into the 1st textbox and hit set offer
  • Now a Answer SDP will be generated in the second text box copy it
  • Open client 1
  • Now paste the Answer SDP into the first text box and hit set offer
